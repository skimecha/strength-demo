# Strength Tracker Demo

Interactive demo of Strength Tracker, a mobile-first fitness tracking PWA built by [Matt Breckenridge](https://mattbreckenridge.com).

This is a standalone demo build: no backend, no accounts. It boots pre-loaded with eight weeks of generated sample history (workouts, cardio, nutrition, and bodyweight), and anything you log persists only to your browser's localStorage. The production app runs on Supabase auth and PostgreSQL.

**Try it:** [skimecha.github.io/strength-demo](https://skimecha.github.io/strength-demo/)

Vanilla JavaScript, single HTML file, no build step.
